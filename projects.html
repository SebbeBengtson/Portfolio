Projects

Here are some projects that I have created.

Dungeon Crawler

The Dungeon Crawler projet is a console program that I made, you can move through a world and interact with it using text inputs, a lot was done through object oriented programming. This project was made using C# and Visual Studio 2022.

Immersion

Immersion is a game that i created in Unity, coded in C#, it is a 2D platformer game with philosophical undertones. This project was coded with C#, assembled through Visual Studio 2022 and the visuals were made using Unity.

Pandemonium

Pandemonium is a social media platform, it's main aim is to induce impulsivenes and toxic behavior, users can write responses to random posts that appear by spinning a wheel. This Website was made using HTML, CSS, JavaScript and PHP for coding, Laragon and HeidiSQL for the database and loading the site.

Fractal Project

A program that I made that generates a fractal and Julia patern with different adjustable settings, achieved through painting the fractal or Julia equations out in the box. This program was made using Windows Forms and coded with C#.
